วิธีติดตั้ง Rich Menu

1) เปิด LINE Official Account Manager หรือใช้ Messaging API
2) สร้าง Rich Menu ขนาด 2500 x 1686
3) อัปโหลดไฟล์ภาพ kck-rich-menu.png
4) ใช้ค่าในไฟล์ kck-rich-menu.json ตั้งค่าพื้นที่กดแต่ละปุ่ม

ปุ่มทั้ง 6:
- บริจาคตอนนี้ → เปิด projects.html?case=donate
- ดูเคสช่วยเหลือ → เปิด projects.html
- โครงการของเรา → เปิด cases.html
- ติดต่อเจ้าหน้าที่ → ส่งข้อความ "ติดต่อเจ้าหน้าที่"
- ขอความช่วยเหลือ → ส่งข้อความ "ขอความช่วยเหลือ"
- เกี่ยวกับมูลนิธิ → ส่งข้อความ "เกี่ยวกับมูลนิธิ"

ถ้าต้องการ ผมสามารถทำ server.js เวอร์ชันรองรับปุ่มทั้ง 3 ข้อความให้อีกชุดได้
